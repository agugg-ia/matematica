El Cuadrado de un Binomio

En esta presentación descubriremos uno de los conceptos más importantes del álgebra: el cuadrado de un binomio.

Aprenderemos a desarrollar expresiones como (a + b)² de manera algebraica y visual, comprendiendo el origen de la fórmula que utilizamos.

(a + b)² = a² + 2ab + b²

Esta fórmula es fundamental para:

  • Simplificar expresiones algebraicas
  • Resolver ecuaciones cuadráticas
  • Comprender conceptos geométricos
  • Aplicaciones en física y estadística

¿Qué es un binomio?

Un binomio es una expresión algebraica que consta de dos términos separados por un signo de suma (+) o resta (-).

Ejemplos de binomios:

a + b
3x - 5
2m² + 7n

Cuando elevamos un binomio al cuadrado, estamos multiplicándolo por sí mismo:

(a + b)² = (a + b) × (a + b)

Para entender cómo se desarrolla esta expresión, podemos multiplicar término por término, aplicando la propiedad distributiva.

Desarrollo Algebraico

Vamos a desarrollar (a + b)² multiplicando término por término:

1
Escribimos la multiplicación del binomio por sí mismo:
(a + b)² = (a + b) × (a + b)
2
Aplicamos la propiedad distributiva (también llamada FOIL):
= a × a + a × b + b × a + b × b
3
Realizamos las multiplicaciones:
= a² + ab + ba + b²
4
Como ab y ba son iguales (propiedad conmutativa):
= a² + 2ab + b²
(a + b)² = a² + 2ab + b²

¡Esta es la fórmula del cuadrado de un binomio! Observa que NO es simplemente a² + b², sino que incluye el término 2ab.

El cuadrado grande tiene lado a + b, por lo que su área es: (a + b) × (a + b) = (a + b)²

Esta área se compone de:

  • Un cuadrado de lado a: área = a²
  • Un cuadrado de lado b: área = b²
  • Dos rectángulos de lados a y b: áreas = ab + ba = 2ab

Observa cómo cada lado del cuadrado grande está dividido en dos segmentos: uno de longitud a y otro de longitud b.

Área total = a² + ab + ba + b² = a² + 2ab + b²

Así visualizamos claramente que (a + b)² ≠ a² + b², sino que debemos incluir los dos rectángulos de área ab.

Ejemplos Numéricos

Ahora apliquemos la fórmula con números concretos para comprender mejor:

Ejemplo 1: (x + 3)²

Aplicamos la fórmula: (a + b)² = a² + 2ab + b²

Donde a = x y b = 3:

(x + 3)² = x² + 2·x·3 + 3² = x² + 6x + 9

Ejemplo 2: (2y + 5)²

Donde a = 2y y b = 5:

(2y + 5)² = (2y)² + 2·(2y)·5 + 5² = 4y² + 20y + 25

Ejemplo 3: (4 - z)²

Podemos escribir (4 - z)² como (4 + (-z))², entonces a = 4 y b = -z:

(4 - z)² = 4² + 2·4·(-z) + (-z)² = 16 - 8z + z²

Nota: (-z)² = z² porque cualquier número al cuadrado es positivo.

Observa cómo en el último ejemplo, con un signo negativo, el término del medio (2ab) se convierte en negativo, pero el último término (b²) siempre es positivo.

Fórmula General para Binomios

La fórmula del cuadrado de un binomio funciona tanto para suma como para resta:

(a + b)² = a² + 2ab + b²

(a - b)² = a² - 2ab + b²

La única diferencia es el signo del término 2ab.

En general, podemos escribir:

(a ± b)² = a² ± 2ab + b²

Donde el signo de 2ab coincide con el signo entre a y b en el binomio original.

También podemos aplicar la fórmula a binomios con coeficientes:

(ma + nb)² = m²a² + 2mnab + n²b²

Por ejemplo: (3x + 4y)² = 9x² + 24xy + 16y²

Ejercicio 1: (x + 7)² = ?

Ejercicio 2: (2a - 5)² = ?

(2a - 5)² = (2a)² + 2·(2a)·(-5) + (-5)² = 4a² - 20a + 25

Ejercicio 3: (3x + 2y)² = ?

(3x + 2y)² = (3x)² + 2·(3x)·(2y) + (2y)² = 9x² + 12xy + 4y²

Ejercicio 4: (m - 4n)² = ?

(m - 4n)² = m² + 2·m·(-4n) + (-4n)² = m² - 8mn + 16n²

Recuerda: El error más común es olvidar el término 2ab y escribir solamente a² + b². ¡No caigas en esa trampa!

Conclusión y Aplicaciones

Hemos aprendido que el cuadrado de un binomio (a + b)² se desarrolla como:

(a + b)² = a² + 2ab + b²

Puntos clave para recordar:

  • El término 2ab es fundamental y no puede omitirse
  • La fórmula funciona tanto para suma como para resta
  • Podemos comprenderla algebraicamente y geométricamente
  • Es aplicable a expresiones con coeficientes y múltiples variables

Aplicaciones del cuadrado de un binomio:

  • Resolver ecuaciones cuadráticas completando el cuadrado
  • Simplificar expresiones en cálculo y álgebra avanzada
  • Cálculo de áreas y volúmenes en geometría
  • Desarrollar fórmulas en física (cinemática, energía, etc.)
  • Análisis estadístico (varianza y desviación estándar)
