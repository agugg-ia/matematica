1. 📶 Servicio de internet

Cargo fijo $1500 (primeros 50 GB) + $20 por GB adicional.

📋 Tabla de valores

Consumo (GB)030506080100
Costo ($)150015001500170021002500
Relleno: hasta 50 GB solo fijo. Para 60: 1500 + 20·(60-50) = 1700; 80: 1500+20·30 = 2100; 100: 1500+20·50 = 2500.

📐 Fórmula del costo

\[ C(x) = \begin{cases} 1500 & \text{si } 0 \le x \le 50 \\ 1500 + 20\,(x-50) & \text{si } x > 50 \end{cases} \]

💰 Si pagaron $1900, ¿cuántos GB?

Usamos la segunda rama: \(1500 + 20(x-50) = 1900\) → \(20(x-50) = 400\) → \(x-50 = 20\) → x = 70 GB.

📈 Gráfica (función partida)

La línea horizontal hasta 50 GB, luego pendiente 20.

2. 🥟 Venta de empanadas

Ganancia diaria: \(G(x) = -x^{2} + 120x - 2000\)

📊 Evaluaciones

x (empanadas)20406080100
G(x) ($)01200160012000
\(G(20) = -400 + 2400 - 2000 = 0\)
\(G(40) = -1600 + 4800 - 2000 = 1200\)
\(G(60) = -3600 + 7200 - 2000 = 1600\)
\(G(80) = -6400 + 9600 - 2000 = 1200\)
\(G(100) = -10000 + 12000 - 2000 = 0\)

🏆 Ganancia máxima (vértice)

\(x_v = -\dfrac{b}{2a} = -\dfrac{120}{2\cdot(-1)} = 60\) empanadas.
\(G(60) = 1600\) → máxima ganancia $1600.

📉 Cantidad para no perder (G ≥ 0)

Resolvemos \( -x^2+120x-2000 = 0\) → \(x^2 -120x +2000 = 0\).
\(\Delta = 120^2 - 4\cdot2000 = 14400 - 8000 = 6400\), \(\sqrt{\Delta}=80\).
Raíces: \(x = \frac{120 \pm 80}{2} = 20\) y \(100\).
Como la parábola abre hacia abajo, \(G \ge 0\) entre las raíces: \(20 \le x \le 100\). Mínimo para no perder: 20 empanadas.

3. 🏀 Partido de básquet

\(h(t) = -5t^{2} + 10t + 2\) (t seg, h metros)

📌 Altura de lanzamiento

\(h(0) = 2\) m

⏫ Altura máxima e instante

\(t_v = -\dfrac{10}{2\cdot(-5)} = 1\) segundo.
\(h(1) = -5(1)^2 + 10(1) + 2 = 7\) metros.

⏬ ¿Cuándo toca el suelo? (h=0)

\( -5t^2+10t+2 = 0\) → \(5t^2 -10t -2 = 0\).
\(\Delta = (-10)^2 -4\cdot5\cdot(-2) = 100 + 40 = 140\), \(\sqrt{\Delta}\approx 11.832\).
\(t = \frac{10 + 11.832}{10} \approx 2.183\) segundos (valor positivo).

⏳ Tiempo en el aire

\(\approx 2.18\) s

4. 📉 Función definida

Dominio \([-3,5]\); puntos: \((-3,1) \rightarrow (-1,4) \rightarrow (2,-2) \rightarrow (5,3)\).

✏️ Gráfica con segmentos

🔺 Máximos y mínimos

Máximo absoluto: en \(x = -1\), valor \(4\).
Mínimo absoluto: en \(x = 2\), valor \(-2\).

🌐 Dominio e imagen

Dominio: \([-3,5]\) (dado).
Imagen: al ser continua (segmentos), el conjunto de valores va desde \(-2\) hasta \(4\) → \([-2,4]\).

📈 Intervalos de crecimiento/decrecimiento

Crece en \([-3,-1]\) y \([2,5]\); decrece en \([-1,2]\).

5. 🔎 Verdadero o Falso

1
Verdadero Una función puede tener tramos crecientes y decrecientes (p. ej., la del ejercicio 4 o cualquier cuadrática).
2
Falso El punto más alto se llama máximo (absoluto o relativo). "Vértice" es exclusivo de parábolas.
3
Verdadero El dominio es el conjunto de valores que puede tomar la variable independiente según el contexto (ej: consumo no negativo).
4
Falso Una función lineal puede ser decreciente (pendiente negativa) o constante, no siempre creciente.
5
Verdadero Toda función cuadrática tiene como gráfica una parábola (abre hacia arriba o abajo).
